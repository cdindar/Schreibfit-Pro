# Schreibfit-Pro
Github Repository für unser Projekt (Rechtschreibtrainer, Kooperation ITP/SEW)

Es gibt Trainingsarten wie Doppelkonsonanten, Umlaute etc und auch einen Nomen Trainer der dafür sorgen soll dass man sich Groß-Kleinschreibung in hinsicht auf Nomen gut einprägt. Außerdem haben wir noch einen Lückentext wo man die Lücke füllen muss, entweder in Form von fertigen buttons oder durch das selber eintippen von Buchstaben
