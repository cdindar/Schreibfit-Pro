# Schreibfit-Pro
Github Repository für unser Projekt (Rechtschreibtrainer, Kooperation ITP/SEW)

Zuerst kommt eine Anmelde-Page wo lediglich nur überprüft wird ob man eine TGM-Schüler E-Mail-Adresse eingibt. Als Passwort kann man irgendwas eingeben da keine Benutzer gespeichert werden. Es ist lediglich nur ein "Mini-Konzept". Es gibt Trainingsarten wie Doppelkonsonanten, Umlaute etc und auch einen Nomen Trainer der dafür sorgen soll dass man sich Groß-Kleinschreibung in hinsicht auf Nomen gut einprägt. Außerdem haben wir noch einen Lückentext wo man die Lücke füllen muss, entweder in Form von fertigen buttons oder durch das selber eintippen von Buchstaben
